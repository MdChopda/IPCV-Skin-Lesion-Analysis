# IPCV-Skin-Lesion-Analysis
Final project for Image Processing and Computer Vision (IPCV). Automated melanoma classification using DullRazor, CLAHE, and a fine-tuned DenseNet121 architecture.

# Melanoma Classification using DenseNet121

This repository contains the complete image processing and deep learning pipeline for my IPCV final project.

**Note on Data:** The code is designed to run on the publicly available HAM10000 dataset. Due to file size limitations, the raw images are not hosted in this repository. To execute the notebook, please download the HAM10000 dataset and update the local directory paths in Step 2 of the Jupyter Notebook.
